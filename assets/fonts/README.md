# Roboto Fonts Required

Please download and place the following font files in this directory:

1. **Roboto-Regular.ttf**
2. **Roboto-Medium.ttf**
3. **Roboto-Bold.ttf**

## Quick Download

Visit: https://fonts.google.com/specimen/Roboto

Click "Download family" and extract:
- Roboto-Regular.ttf
- Roboto-Medium.ttf 
- Roboto-Bold.ttf

Place them in this `assets/fonts/` directory.

## Alternative

The app will work without fonts but won't match the intended design.
To run without fonts, see FONTS_SETUP.md in the project root.
