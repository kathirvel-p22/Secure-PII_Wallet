## 📱 First Use

1. **Launch App** → Splash screen appears
2. **Tap UNLOCK** → Enter dashboard
3. **Tap UPLOAD** → Add your first file
4. **Choose Mode** → Secure or High Secure
5. **Set Password** → Strong password required
6. **Generate Keys** → (High Secure mode only)
7. **Upload** → File encrypted and stored
8. **View File** → Tap file, enter credentials
9. **Enjoy!** → Your data is secure
